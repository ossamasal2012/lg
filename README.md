# OSSIPTV - LG webOS Media Player

**OSSIPTV** هو مشغل وسائط احترافي وخفيف الوزن مصمم خصيصاً لشاشات LG Smart TV التي تعمل بنظام webOS. يهدف التطبيق إلى توفير تجربة مشاهدة سلسة وبسيطة للمستخدمين لتشغيل قوائم التشغيل الخاصة بهم.

## المميزات
* **تصميم مريح:** واجهة مستخدم باللون الأسود الملكي (#000000) مثالية لشاشات OLED.
* **دعم البث:** مهيأ لتجاوز قيود CORS لضمان استقرار روابط البث المباشر.
* **دقة عالية:** يدعم العرض بدقة Full HD 1920x1080.
* **خفيف وسريع:** يعتمد على تقنيات الويب الحديثة لضمان سرعة الاستجابة.

## كيفية الاستخدام
التطبيق يعمل كمشغل (Player) فقط. يجب على المستخدم توفير روابط البث الخاصة به (M3U8/HLS) عبر الواجهة البرمجية المدمجة.

## سياسة الخصوصية (Privacy Policy)
تطبيق OSSIPTV لا يقوم بجمع أو تخزين أو مشاركة أي بيانات شخصية خاصة بالمستخدمين. التطبيق هو مجرد أداة لعرض المحتوى الذي يوفره المستخدم ولا يتتبع سجل المشاهدة.
